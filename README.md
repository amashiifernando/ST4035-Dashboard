# Binary Logistic Regression Interactive Dashboard

## Dashboard Preview

![Dashboard Preview](dashboard.png)

## Overview

This project is an interactive web-based dashboard developed for the ST 4035 Data Science course. The dashboard demonstrates key concepts of Binary Logistic Regression, including class imbalance handling, balancing techniques, link functions, ROC analysis, and confusion matrix evaluation.

The dashboard is built entirely using HTML, CSS, JavaScript, and Plotly.js and runs directly in a web browser without requiring any server-side code.

---

## Features

### Interactive Controls

* Sample Size Slider (100–2000 observations)
* Minority Class Percentage Slider (5%–50%)
* Balancing Method Selection

  * None (Imbalanced)
  * Oversampling
  * Undersampling
  * SMOTE
* Link Function Selection

  * Logit
  * Probit
  * Complementary Log-Log (Cloglog)

### Live Metrics Panel

* Total Samples
* Class 1 Count (Minority Class)
* Class 0 Count (Majority Class)
* Imbalance Ratio

### Interactive Visualizations

1. Class Distribution Before and After Balancing
2. Fitted Probability Curves for Different Link Functions
3. ROC Curve with AUC Calculation
4. Normalized Confusion Matrix Heatmap

### Performance Summary

* AUC
* Accuracy
* Sensitivity (TPR)
* Specificity (TNR)
* Active Link Function
* Active Balancing Method
